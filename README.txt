Credit to https://github.com/corkami/collisions for the PNG collision code (png.py, png1.bin, png2.bin)

Pre-requisites: install python, have python in your PATH variable

Usage: python main.py file1.png file2.png

Output: MD5 hash and filesize of the original and post collision files, SHA-256 hashes of the MD% colliding files

dog.png and fish.png have been provided for easy testing
